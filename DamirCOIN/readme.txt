Развлекательный сайт кликер Дамбакоин.
Страницы:
	1. Главая страница из которой можно попасть в другие. На главной используется API - шутки Чака Норисса.
	2. Регистрация - страница, в которой можно вписать Имя, эл. почту, пароль. Минимальная длинна пароля - 8 симв. Проверка всегоо проходит. после успешной регистрации данные попадают в БД.
	3. Логин - страница входа, вводятся данные, сравниваются с БД и в случаем успешного совпадения, пересылает на кликер.
	4. Кликер - страница, где есть большая кнопка для набратия коинов и улучшения которые можно покупать за эти койны. Данные в БД передаются каждую секунду через JS.
	5. Топ-лист - страница, где видно 10 юзеров с наибольшим кол-вом коинов.
	6. Донат - страница, где после вписания суммы, можно её задонатить. Вас пересылает через API на страница с оплатой.
Лицо на страницах принадлежит соавтору (дизайнеру) - Дамиру Васильеву.
